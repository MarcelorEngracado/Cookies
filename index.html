📚 O que são Cookies?

Cookies são pequenos arquivos de texto armazenados no navegador do cliente. Eles permitem que websites "lembrem" informações sobre o usuário.

Tamanho: ~4KB por cookie
Limite: ~180 cookies por site
Acesso: Apenas via JavaScript ou cabeçalho HTTP
Segurança: Visíveis ao cliente (não criptografados por padrão)

💻 Código JavaScript Explicado

1️⃣ Criar um Cookie

function setCookie(nome, valor, dias) {
    let data = new Date();
    data.setTime(data.getTime() + (dias * 24 * 60 * 60 * 1000));
    let expira = "expires=" + data.toUTCString();
    document.cookie = nome + "=" + valor + ";" + expira + ";path=/";
}

2️⃣ Ler um Cookie

function getCookie(nome) {
    let nomeComIgual = nome + "=";
    let cookies = document.cookie.split(';');
    for(let i = 0; i < cookies.length; i++) {
        let cookie = cookies[i].trim();
        if (cookie.indexOf(nomeComIgual) === 0) {
            return cookie.substring(nomeComIgual.length);
        }
    }
    return null;
}

3️⃣ Deletar um Cookie

function deleteCookie(nome) {
    setCookie(nome, "", -1);
}

⚠️ Dicas de Segurança

  • Não armazene dados sensíveis: Senhas, tokens, dados bancários não devem ir em cookies simples
  • Use HTTPS: Sempre transmita cookies via conexão segura
  • HttpOnly flag: Em cookies servidor, use a flag HttpOnly para evitar acesso via JavaScript
  • SameSite: Proteja contra ataques CSRF usando o atributo SameSite
  • Expiração: Sempre defina uma data de expiração apropriada
